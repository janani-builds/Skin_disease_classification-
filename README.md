✨ DermAI – Intelligent Skin Disease Detection System

DermAI is an AI-powered web application that detects different types of skin diseases using Deep Learning and Computer Vision techniques. Users can upload a skin image, and the system predicts the possible skin disease using a trained CNN model.

---

🚀 Features

- AI-powered skin disease prediction
- Deep Learning CNN model
- Modern and responsive frontend UI
- Image upload functionality
- Flask backend integration
- Real-time prediction system

---

🛠️ Technologies Used

- Python
- Flask
- TensorFlow / Keras
- HTML
- CSS
- JavaScript
- OpenCV
- NumPy

🧠 Model Training

The CNN model was trained using a skin disease image dataset. The model processes uploaded images and predicts the disease category.

⚙️ Installation

1. Clone Repository

git clone https://github.com/your-username/DermAI.git

2. Open Project Folder

cd DermAI

3. Create Virtual Environment

python -m venv venv

4. Activate Virtual Environment

Windows

venv\Scripts\activate

---

📦 Install Dependencies

pip install -r requirements.txt

---

▶️ Run Application

python app.py

Open browser:

http://127.0.0.1:5000
